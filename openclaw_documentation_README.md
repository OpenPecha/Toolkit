## Features

- Create, edit, and export Tibetan texts
- Annotation support
- Multiple output formats
- OpenPecha format integration

## Prerequisites

- Python 3.8+

## Installation

```bash
git clone https://github.com/OpenPecha/Toolkit.git
cd Toolkit
pip install -e .
```

## Usage

```python
from openpecha import toolkit

# Tool usage examples
toolkit.create_text(...)
toolkit.export_text(...)
```
